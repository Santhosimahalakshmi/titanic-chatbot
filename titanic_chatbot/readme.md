# Titanic Chatbot 🚢

This is a FastAPI-based chatbot built using the Titanic dataset.

## Features
- FastAPI backend
- Swagger API testing
- Basic chatbot response
- Ready for deployment

## Run Locally

1. Create virtual environment
2. Install dependencies
3. Run:

uvicorn main:app --reload

Open:
http://127.0.0.1:8000/docs
